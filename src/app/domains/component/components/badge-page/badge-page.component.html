Badges are indicators of static information. They can be updated when a status changes, but they should not be actionable. Default Use a default badge for basic information that not need attention.
the default form of a badge
Primary Use a primary badge to help draw attention to new or updated information.
Icon Badge Use an icon to give more information about the badge.
the form of an icon badge
Status Badge Use different color for each different status.

Angular CLI ng g c "your desired path"/badge
